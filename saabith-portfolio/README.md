This repository contains a personal portfolio built with React, Tailwind CSS and Framer Motion. The site showcases projects, core skills, and contact information.

Features
- Modern, responsive design with colorful gradients and subtle animations
- Sections: About, Projects, Skills, Contact
- Links to GitHub, LinkedIn and email

Running locally
1. Install dependencies:

```powershell
npm install
```

2. Start the development server:

```powershell
npm start
```

Open http://localhost:3000 to view the portfolio.
